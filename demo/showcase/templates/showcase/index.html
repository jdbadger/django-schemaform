Django Forms from Pydantic Models
What is Django SchemaForm?

SchemaForm is a Django form class that automatically creates form fields from Pydantic schemas. It handles type mapping, validation, constraints, and more‚Äîall from your Pydantic model definitions.

Try it out: Click on any form below to see the Pydantic schema alongside the generated form.

Key Features

üîÑ Type Mapping

Python types automatically map to Django form fields: str ‚Üí CharField, EmailStr ‚Üí EmailField, SecretStr ‚Üí Password input, etc.

‚úÖ Pydantic Validation

All Pydantic validation rules are enforced with user-friendly error messages. Supports Field(min_length=..., ge=..., le=...) and more.

üé® HTML5 Widgets

Date, time, datetime fields use native HTML5 pickers. FutureDate and PastDate set min/max attributes automatically.

üìù Field Metadata

Field(description=...) becomes help text. Field(title=...) becomes the label. Docstrings are preserved.
